# Product Scope Status

The attached Deep Research blueprint is treated as the product specification. This repository implements the feasible local CPU playground scope:

- Repository scaffold
- TypeScript frontend source and local FastAPI backend
- Blossom and frosted-glass visual shell
- Accessibility foundations and reduced-motion support
- Shared playground and model-adapter contracts
- Complete Digit Lab vertical slice
- Image Repair Studio surface with an explicit licensed-checkpoint blocker
- Emotion/Text Lab using local SST-2 sentiment inference
- Sound Lab using local YAMNet audio classification
- Style and Colour Studio using local ONNX Fast Neural Style models
- Model Battle endpoints for Digit, Text, Sound, and Style
- Curiosity challenges and simple/intermediate/advanced explanations
- Local experiment saving in browser storage
- Real local ONNX model files and offline operation after setup
- Tests for preprocessing, adapters, API behavior, and critical UI expectations

The only unresolved blueprint requirement is a packaged pretrained autoencoder. The researched checkpoints did not expose compatible licence metadata, so the application does not simulate image repair or package those weights.
